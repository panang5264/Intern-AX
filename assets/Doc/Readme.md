## ⚙️ การทำงานใน Cocos Creator
- **โครงสร้างไฟล์**: ทุก asset และ script ต้องอยู่ภายใต้โฟลเดอร์ `assets`  
- **การแยกโค้ด**: สร้างโฟลเดอร์ย่อย เช่น `GameAssets`, `GameCode`, `Docs` ภายใน `assets` เพื่อแยกหมวดหมู่  
- **ไฟล์ .meta**:  
  - ต้อง commit และ push ขึ้น Git พร้อมกับ asset ทุกครั้ง  
  - ไม่ควร ignore เพราะไฟล์ `.meta` เก็บข้อมูลสำคัญ เช่น UUID และการตั้งค่า import ของ asset  
  - หากไม่มีไฟล์ `.meta` เวลาเพื่อน pull โปรเจกต์ไปทำต่อ asset จะสูญเสียการอ้างอิง ทำให้ prefab หรือ scene ใช้งานไม่ได้  
  - เพื่อให้ทีมสามารถทำงานต่อเนื่องได้อย่างราบรื่น ควร push ไฟล์ `.meta` เสมอ
